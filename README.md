### Install
```
git clone https://github.com/joaostack/SimpleMalObfuscator; cd SimpleMalObfuscator
```

```
dotnet restore
```

```
dotnet publish
```

And go to "bin" and "Release" folder.

Sent .ps1 file to victim.
